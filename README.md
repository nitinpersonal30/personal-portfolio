## 🚀 How to Use

1. Download or clone this repository.
2. Open `index.html` in any browser.
3. Customize the content according to your details.
4. Upload the project to GitHub.
5. Enable GitHub Pages from repository settings.

---

## 🌐 Deployment Using GitHub Pages

To make this website live:

1. Go to repository **Settings**
2. Click on **Pages**
3. Select branch: `main`
4. Select folder: `/root`
5. Click **Save**

Your portfolio website will be live after a few minutes.
